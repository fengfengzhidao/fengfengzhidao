## 官网

[枫枫知道官网](https://www.fengfengzhidao.com/) 是我的个人技术博客和课程内容入口，主要记录后端、前端、架构与 AI 相关的学习笔记、项目实践和工具实验。

官网主要功能：

- 技术文章：持续更新 Go、Vue、Python、Codex、AI 工具、项目部署等方向的文章。
- 学习路线：整理编程开发路线和 AI 应用路线，覆盖 Git、MySQL、Linux、Go、Gin、GORM、Vue3、Redis、微服务、Docker、Nginx 等内容。
- 专题文档：按系列汇总 Go 基础、Gin、GORM、MySQL、Linux、gRPC、Vue3、Nuxt、go-zero、Codex 新手教程等专题内容。
- AI 产品：展示我用 AI 做出来的产品、工具和实验项目，例如枫枫知道生图、枫枫笔记、公众号 Markdown 发布工具、Codex 工具箱、资源导航和大模型网络测试工具。
- 课程入口：提供 Go 全栈课、AI 专题课等课程信息，方便系统化学习和项目实战。
- 站内能力：支持文章搜索、用户登录、在线反馈、标签云、打赏列表和社交平台入口。
